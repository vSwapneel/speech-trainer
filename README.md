# Speech Trainer

A voice training app that helps users speak with confidence by analyzing their speech in real-time and providing personalized exercises with natural, expressive feedback.

## Features

- Real-time speech analysis using **Vertex AI / Gemini**
- Transcription and TTS feedback via **ElevenLabs**
- Tracks core metrics:  
  - Filler words  
  - Pitch / voice shakiness  
  - Speaking pace  
- Personalized exercises to improve confidence and authority in speech
- Dashboard to monitor progress over time
